什么是Web？

web是互联网的总称，全称为World Wide Web，缩写WWW ，即全球广域网，也称为万维网，它是一种基于超文本和HTTP的、全球性的、动态交互的、跨平台的分布式图形信息系统。简单地说，Web是一种体系结构，通过它可以访问遍布于因特网（Internet）主机上的链接文档。

Web 有什么特点

（ 1 ）一体性

Web 非常流行的一个很重要的原因就在于它可以在一页上同时显示色彩丰富的图形和文本的性能。在 Web 之前 Internet 上的信息只有文本形式。 Web 可以提供将图形、音频、视频信息集合于一体的特性。

（ 2 ）独立性

无论用户的系统平台是什么，你都可以通过 Internet 访问 WWW ，浏览 WWW 对系统平台没有什么限制。

（ 3 ）交互性

Web 的交互性首先表现在它的超链接上，用户的浏览顺序和所到站点完全由他自己决定。另外通过 FORM 的形式可以从服务器方获得动态的信息。用户通过填写 FORM 可以向服务器提交请求，服务器可以根据用户的请求返回相应信息。

WEB一些基本术语

Internet：前身是阿帕奇网，把多台电脑终端连接在一起

现在成为广域网（Internet）:互联网

WWW（万维网）提供一定的网页浏览服务：网上购物，网络共享，为我们提供网页浏览服务

URL（统一资源定位符）：就是俗称的网址：如https://www.youtube.com/

DNS（域名解析）：每一个主机都有一个世界上唯一的IP

IP 地址和我们的网页是一一对应的。如：202.188.0.133是TMNet的IP地址

https://www.youtube.com/ DNS就是把我们的域名解析成为IP地址的过程

HTTP（超文本传输协议）：做了一些通讯的规则。

想把网站连到互联网上就要遵循互联网上的协议（HTTP）

W3C（万维网联盟）：一个组织，制作WEB语法规范
